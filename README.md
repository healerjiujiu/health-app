# 糖尿病照护提醒 App（PWA 本地网页版）

这是一个可离线运行、可安装到手机桌面的提醒应用，包含：

- 喝水提醒（间隔可配置、喝水打卡）
- 糖尿病饮食提醒（早餐/午餐/晚餐/加餐）
- 高风险警醒模式（对不执行行为升级提醒频率）
- 今日记录（不喝酒、不吸烟、血糖监测、喝水执行）

## 使用方法

1. 用浏览器打开 `index.html`。
2. 点击“开启通知”并允许通知权限。
3. 点击“安装到手机桌面”（iPhone 可在分享菜单里“添加到主屏幕”）。
4. 设置喝水提醒与饮食提醒时间。
5. 建议保持高风险警醒模式开启。

## 高风险警醒模式说明

- 会根据当天执行情况生成风险评分。
- 当出现以下情况会更频繁提醒：
  - 长时间未喝水
  - 未监测血糖
  - 未勾选禁酒、禁烟
  - 未确认当天警醒
- 出现明显不适时，可点击“出现不适，立即就医提醒”并及时就医。

## 医疗提示

- 应用仅用于生活提醒，不替代医生诊疗。
- 血糖显著升高或尿酮体阳性伴随不适时，应及时就医。
